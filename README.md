# 🛒 **E-Commerce REST API**

## 🚀 **Secure Spring Boot Backend for an E-Commerce Platform**

This project provides REST APIs for **user authentication, product management, shopping cart operations, and order processing** using **Java, Spring Boot, Spring Security, and JWT**.

---

# 🧰 **Tech Stack**

* ☕ **Java**
* 🌱 **Spring Boot**
* 🔐 **Spring Security + JWT**
* 🗄 **MySQL**
* 📦 **Maven**
* 🧪 **Postman**
* 🐙 **Git & GitHub**

---

# ✨ **Features**

## 👤 **User Module**

* 📝 **User Registration**
* 🔑 **JWT Login Authentication**
* 👥 **Role-Based Access (Admin / Customer)**

## 📦 **Product Module**

* ➕ **Add Products (Admin)**
* ✏ **Update Products**
* ❌ **Delete Products**
* 🔍 **Search Products**
* 📋 **View Product Catalog**

## 🛒 **Cart Module**

* ➕ **Add Items to Cart**
* 🔄 **Update Quantity**
* ❌ **Remove Items**
* 👀 **View Cart**

## 📑 **Order Module**

* 🧾 **Place Orders**
* 📚 **Order History**
* 🚚 **Track Order Status**
